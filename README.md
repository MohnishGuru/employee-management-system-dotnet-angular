# Employee Management System (Angular + ASP.NET Core)

##  Overview
A full-stack Employee Management System developed using Angular and ASP.NET Core Web API.
The application allows managing employee records with secure and scalable RESTful APIs.

## Features
- Employee CRUD Operations
- RESTful APIs using ASP.NET Core Web API
- Angular-based responsive UI
- SQL Server database integration
- Role-based access and validation
- Clean architecture and modular code

## Technologies Used
### Frontend
- Angular
- TypeScript
- HTML5
- CSS3
- Bootstrap

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- LINQ

### Database
- SQL Server
